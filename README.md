# 🌾 Agromitra Admin Dashboard

Agromitra Admin Dashboard is a web-based admin panel designed to manage and monitor the Agromitra platform.  
The system enables administrators to efficiently manage users, analyze agricultural data, and monitor platform activities.

The project was developed as part of **Smart India Hackathon 2024 (Finalist Project)** aimed at helping farmers make data-driven decisions for better crop yield and fertilizer optimization.

---

## 🚀 Features

- Admin authentication and secure login
- Dashboard with analytics and statistics
- Farmer and user management
- Data monitoring for agricultural insights
- Interactive charts and reports
- Responsive admin interface

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Ayushigarg24/AgromMitraAdmin.git
```

---

### 2️⃣ Navigate to the project directory

```bash
cd AgromMitraAdmin
```

---

### 3️⃣ Install dependencies

```bash
npm install
```

---

### 4️⃣ Start the development server

```bash
npm start
```

The application will run at:

```
http://localhost:3000
```

## 📊 Project Purpose

Agromitra was developed to assist farmers with **data-driven insights** related to:
